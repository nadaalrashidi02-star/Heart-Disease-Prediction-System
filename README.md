# 📘 Heart Disease Prediction (Data Mining Project)

## 📌 Overview

This project applies data mining and machine learning techniques to analyze medical data and predict the likelihood of heart disease.
It covers the full pipeline including data preprocessing, exploratory data analysis (EDA), model building, and performance evaluation.

---

## ⚙️ Project Workflow

### 1️⃣ Data Preparation

* Handling missing values (mean, median, mode)
* Removing columns with high missing rates
* Data type conversion (string to numeric)
* Standardizing categorical variables (e.g., gender)
* Removing duplicates
* Outlier detection and handling (IQR method)
* Data transformation (unit conversion and encoding)

---

### 2️⃣ Exploratory Data Analysis (EDA)

* Distribution analysis (histograms)
* Outlier detection (boxplots)
* Feature relationships (scatter plots)
* Identifying important health indicators

---

### 3️⃣ Data Splitting

* Training set: 70%
* Testing set: 30%

---

### 4️⃣ Machine Learning Models

The following models were implemented:

* Random Forest
* Decision Tree
* Logistic Regression
* K-Nearest Neighbors (KNN)

---

### 5️⃣ Model Evaluation

Models were evaluated using:

* Accuracy
* Precision
* Recall
* F1-score
* Confusion Matrix

---

### 6️⃣ Model Tuning

* Hyperparameter tuning using GridSearchCV
* Performance comparison before and after tuning

---
